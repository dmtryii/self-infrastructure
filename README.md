# Self infrastructure

A personal repository of cheatsheets, scripts, and practical guides for managing servers, infrastructure, DevOps tasks, and utilities.

## Purpose
- Quick access to commands and configurations for server setup, security, networking, and development environments.
- Store practical examples and solutions for daily work and learning.

## Structure
- `vps/` – server setup, SSH, firewall, utilities.
- `dev/` – development environment setup and code examples.
- `tools/` – scripts and automation commands.
- `notes/` – tips, concepts, and quick reference guides.
